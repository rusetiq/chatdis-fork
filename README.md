# ChatDIS | Dunes International School Digital Assistant

**ChatDIS** (Prototype) is an intelligent, conversational assistant designed to help students, parents, and faculty navigate the official policies and information of **Dunes International School (Abu Dhabi)**.

By leveraging the official School Almanac and Web data, ChatDIS provides instant, accurate answers to common queries, reducing the administrative load and improving accessibility for the school community.
